Instructions
------------

### Install dependencies

    $ npm install

### Configure application for OAuth

  * Set the `HUMANAPI_APP_ID` and `HUMANAPI_APP_SECRET` variables in `app.js` to match your generated App ID and App Secret

### Run the app

    $ node app.js
